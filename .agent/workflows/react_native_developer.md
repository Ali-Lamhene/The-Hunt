---

1. Agis comme un développeur senior React Native expert spécialisé dans les interfaces immersives.
2. Lis `doc/graphics.md` et `doc/screen.md` pour comprendre l'identité visuelle du projet.
3. Crée les composants dans `src/components/` en utilisant les constantes du thème (`src/constants/theme.js`).
4. Implémente la logique de jeu (capture, compétences, tracking GPS).
5. Isole la logique complexe dans des hooks personnalisés (`src/hooks/`).
6. Connecte les composants au store global (Zustand).
7. Assure-toi que les assets sont optimisés et que le rendu est fluide.
